# ReTour Contribution Guide

## Branch Strategy

- main: stable version
- develop: integration branch
- feature branches: individual tasks

## Branch Naming

Use:

feature/<feature-name>

Examples:
- feature/llm-manager
- feature/fastapi-api
- feature/tourism-data
- feature/frontend-ui

## Commit Messages

Use clear messages:

Examples:
- Add FastAPI project structure
- Implement tourism data schema
- Add RAG retrieval module

## Pull Requests

All changes should be reviewed before merging into develop.
